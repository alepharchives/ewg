h1. Erlang Wordlist Generator

A wordlist generator written in Erlang. 
It uses BeepBeep (and Michiweb) as web/restful frontend.

h2. Authors

* Matteo Redaelli: [http://www.redaelli.org/matteo/]

h2. Prerequisites

* Erlang language: download it from http://erlang.org, but most of Linux distributions already include it.
* make
 
h2. Quick start

See the wiki at [http://wiki.github.com/matteoredaelli/ewg]
