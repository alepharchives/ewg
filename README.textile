h1. Erlang Wordlist Generator

A wordlist generator written in Erlang. 
It uses BeepBeep (and Michiweb) as web/restful frontend.

h2. Quick start

* Download ewg from github
* cd ewg
* edit options in src/ewg.hrl
* make clean && make
* ./start_server.sh
* lunch in your preferred broser http://localhost:8000/ewg/
* click on "Start generator"
* see stdout for logs
* see words.txt for geneated words
